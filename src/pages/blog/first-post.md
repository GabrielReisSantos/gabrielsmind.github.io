# Why I rebuilt GabrielsMind from scratch

The old site did the job badly.

It carried old structure, unnecessary weight, and too much friction for a portfolio that should be simple, fast, and clear.

So I rebuilt it with a cleaner stack and a sharper purpose:

- show real projects
- publish useful writeups
- keep the site fast
- reduce cost and maintenance

This version is the foundation, not the final form.
